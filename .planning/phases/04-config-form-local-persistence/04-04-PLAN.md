## Phase Goal

**As a** self-hosting user describing a packing job, **I want to** edit numeric fields, toggle flags, and pick a rotation mode through clear, accessible controls, **so that** I can describe the job precisely.

## Artifacts this phase produces

- **`src/components/NumberField.tsx`:** props for `label`, `unit` (`mm`/`kg`/`pcs`), `hint?`, `error?`, and pass-through input props (so RHF `register(...)` spreads onto the `<input>`). Renders a 600-weight label, a mono 13px value input inside an `input-affix` with the unit affix (mono 12px), an optional hint, and a `--danger` error line below when `error` is set. Focus-within accent ring.
- **`src/components/Switch.tsx`:** controlled `checked` + `onChange(boolean)` + accessible name; `role="switch"`, `aria-checked`; ON track fills with `--color-accent`; supports a `size` prop (full 38×22 / sm 32×18) and a `disabled` state.
- **`src/components/SegmentedControl.tsx`:** generic `value`, `options: {value,label}[]`, `onChange(value)`; radio-group semantics (arrow-key nav, Enter/Space select, exactly one selected); selected segment uses `--accent-weak` bg + `--accent-text`.
- **`src/components/Card.tsx`** and **`src/components/SectionLabel.tsx`:** the card chrome (title/desc/body over `--card-body-padding`, `--radius-lg`, `--shadow`) and the mono section label with the 6px accent leading dot.
