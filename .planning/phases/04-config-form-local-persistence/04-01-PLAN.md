## Phase Goal

**As a** self-hosting user describing a packing job, **I want to** fully describe a pallet and a box catalog in one validated form that survives a refresh, **so that** I can hand a correct request to the packer without signup or losing my work.

(This plan delivers the type/schema/token/dependency seam the rest of the slice is built on; no user-visible capability ships until Wave 3+.)
