Deliver the Explode slider end-to-end as the first user-visible vertical slice: a bottom-center control
that, when dragged above 0, animates the solver's layers apart vertically (D-04 uniform additive gap,
D-07 animated), re-frames the camera to the growing stack (D-05), and hides the now-misleading CoG
marker (D-06) — while at its 0/"Assembled" default the scene is byte-identical to the Phase-6 view (SC-3)
and a pallet switch never moves the camera (D-02 preserved).

result, proving the layer model + the explode mechanics + the camera-decoupling discipline before the
Layers slice adds focus/isolation on top. After this plan a user can do something new: pull a dense
pallet apart layer-by-layer. Implements RESULT-07 (SC-2, SC-3, SC-4 for the preset/heatmap compose;
D-04/05/06/07).
Output: explode offset + animation in Boxes (using the shared EXPLODE_FIXED_UNIT from Plan 01), the
decoupled explode re-fit in CameraPresets, the new LayerControls bottom bar (Explode portion), ResultPage
explode state + wiring + CoG-hide gate, and a route-intercepted explode e2e.
