<objective>
The summary + switcher vertical slice (RESULT-03 + RESULT-04): add the whole-job Summary
rail block (pallets used, utilisation, unpacked, Σ weight — D-03 whole-job scope), the
multi-pallet Switcher (each pallet's label + box count + weight + NEUTRAL fill%, D-04/D-05;
selecting drives the scene), and the computed per-selected-pallet sub-line on the viewer
overlay (D-03). Critically, verify+guard that a pallet swap PRESERVES the camera (D-02,
Pitfall 3) — the existing `CameraPresets` bbox effect lists `bbox` in its deps, so a swap
re-measures the bbox and could re-trigger the preset animation; this slice decouples the
swap from a re-frame and proves it with an e2e camera-position assertion.

After this slice a user can see whole-job stats and click between generated pallets,
seeing each pallet's 3D layout + per-pallet stats, with the camera staying put.

Purpose: deliver SC-2 (summary stats + pallet switching with per-pallet layout/stats).
Output: `SummaryBlock`, `PalletSwitcher` (+ tests), `ViewerOverlay` sub-line, the
camera-preservation guard in `CameraPresets`, the rail wiring in `ResultPage`, and the
pallet-switch e2e (camera unchanged).
</objective>
