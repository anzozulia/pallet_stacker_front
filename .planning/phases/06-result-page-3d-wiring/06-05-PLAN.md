selected pallet's centre-of-gravity in the 3D scene (a marker sphere + a vertical drop-line
to the deck, using the golden `cog.z`-up point-map from Plan 01), toggle-able via a

InstancedMesh (D-12 — keeps per-box emissive hover working).

per-box support diagnostics in the explorable scene.
